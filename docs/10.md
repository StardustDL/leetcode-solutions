# regular-expression-matching

动态规划。`c[i][j]` 表示 `s` 长度为 `i` 的前缀能否匹配 `p` 长度为 `j` 的前缀。

- 边界：`c[0][0]=true`，对于 `i>0`，`c[i][0]=false`
- 对于 `c[i][j]` 进行分类讨论：
  - `p[j]` 是 `*`
    - 选择匹配0个，则为 `c[i][j-2]`
    - 选择匹配多个，则要求 `i>0`，`c[i-1][j]` 且 `s[i]` 能够匹配 `p[j-1]`
  - `p[j]` 不是 `*`，要求 `i>0`，`c[i-1][j-1]` 且 `s[i]` 能够匹配 `p[j]`
- 结果：`c[len(s)][len(p)]`
